**Courier Service Management System API**
This project is a RESTful API for managing courier services, built using Spring Boot and MySQL. It provides basic CRUD functionality for entities like Company, Product, Payment, and more.

*Features*
CRUD Operations: Create, Read, Update, and Delete operations for entities.
Database Integration: Data is stored in a MySQL database with JPA for persistence.
API Testing: Endpoints tested using Postman to ensure functionality.

*Technologies Used*
Java: Primary programming language.
Spring Boot: Framework for building RESTful APIs.
MySQL: Database for storing entity data.
Postman: Tool for testing API endpoints.

*Project Structure*
Controllers: Define REST endpoints for each entity.
Repositories: Handle database operations for each entity.
Models: Represent database tables and manage entity attributes.
